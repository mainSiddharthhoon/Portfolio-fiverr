## 📖 Overview
A premium dark futuristic design, built entirely with vanilla HTML CSS and JavaScript, no frameworks. Featuring a Spline 3D interactive robot, GSAP scroll animations, a liquid glass navbar, an animated loader, and smart device detection to deliver a top-tier visual experience across all platforms.

## ✨ Features
- [x] Cinematic animated loader with progress bar and welcome text
- [x] Liquid glass floating navbar with magnetic blob hover effect
- [x] Interactive Spline 3D robot hero background
- [x] Typing animation cycling through multiple roles
- [x] GSAP ScrollTrigger animations on all sections
- [x] About section with aurora glow photo effect
- [x] Expertise cards with cursor-tracking inner glow
- [x] 6 project showcase with alternating cinematic rows
- [x] Working contact form powered by EmailJS
- [x] Animated favicon that blinks on inactive tab
- [x] Real brand color social media icons
- [x] Smart 3-tier device detection:
  - Desktop → Full interactive Spline 3D
  - Flagship mobile (iOS / Android 8GB+) → Full interactive Spline 3D
  - Budget mobile (Android under 8GB) → Animated CSS aurora fallback
- [x] Fully responsive across all screen sizes
- [x] GPU optimised animations running at 60fps
- [x] Performance optimised for weak devices

## 🛠️ Tech Stack
| Technology | Purpose |
